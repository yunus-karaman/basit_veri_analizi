# basit_veri_analizi
Bu veri analizinde bir e ticaret sitesinin verileri incelenip analiz edilmiştir.
Analizi görüntülemek için yüklenen veri_analizi.docx dökümanına analizin kodları için ise veri_analizi-checkpoint.jpynb dosyasına bakılmalıdır.
Analizi yapılan dosyalar https://www.kaggle.com/datasets/berkayalan/ecommerce-sales-dataset/data sitesinden alınmıştır.
